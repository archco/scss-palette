# scss-palette

The color palette for SCSS.

## Installation

### NPM

``` sh
npm install scss-palette
```

### Bower

``` sh
bower install scss-palette
```

## Usage

### Basic

``` scss
// Import in your style.scss
@import 'node_modules/scss-palette/src/palette';

$ashgray: palette('ashgray');
```
